Проведение разведывательного анализа данных для успешного построения модели, которая предсказывала бы результаты госэкзамена по математике для каждого ученика школы. 

В данных наблюдается равномерное распределение пропущенных значений, что позволяет заполнить их наиболее часто повторяющимся значениями параметра. 
Таким образом сохраняется выборка, размер которой в данном проекте представляет большое значение для построения успешной модели.

Обнаружено незначительное количество выбросов. Два из них находятся в столбцах "образование отца" и "семейные отношения". Но здесь очевидно имеет место быть техническая ошибка при обработке первичных данных.
Поэтому в данном случае мы можем скорректировать значения параметров, чтобы избежать удаления обеих строк и уменьшения выборки.
Удалены лишь два явных вброса для параметра "количество пропущенных занятий". В целом можно сделать вывод о том, что данные достаточно чистые.

Корреляционный анализ для числовых переменных показывает нам наибольшое влияние на итоговые балы по госэкзамену по математике таких переменных как: образование каждого из родителей, а также время на учёбу.
Скорее всего можно говорить о том, что в более образованных семьях дети тратят время на учёбу больше и, соответственно, получают лучше оценки.
Но для дальнейшего анализа и построения модели предлагается использовать параметры даже с незначительной положительной корреляцией: семейные отношения, свободное время после школы, количество пропущенных занятий.

Самые важные параметры, которые предлагается использовать в дальнейшем для построения модели:
Medu, Fedu, studytime, famrel, freetime, absences, sex, address, Mjob, paid, higher, romantic.
